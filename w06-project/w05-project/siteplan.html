Site Plan: Movie Explorer

A web app for discovering movies and exploring film details

Site Name

Movie Explorer

This name reflects the purpose of the site: helping users explore, search, and discover movies. Possible domain: movieexplorer.net

Site Purpose

The website provides a hub for movie lovers to search for films, view ratings, and read details using third-party APIs (OMDb and TMDB). It helps users discover new films, find recommendations, and stay updated with trending movies.

Scenarios

  • Which movies are trending this week?
  • How can I get details about a specific movie (ratings, plot, release year)?

Color Schema

Primary Color: #1E3A8A (used for headings, navigation, accents)

Secondary Color: #F59E0B (used for highlights, buttons, call-to-action)

Typography

  • Roboto – for body text (clean and readable)
  • Lora – for headings (gives a modern and stylish look)
